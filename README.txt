Этот проект - игра устроенная на лад монополии, только для того чтобы успешнее проходить по дороге нужно отвечать
на вопросы определённого поля.
Всего 4 поля: математика, русский, физика, информатика.
Есть возможность поменять цветовую гамму со светлой на темную.

Проект использует несколько ui файлов: для регистрации, рейтинга, авторизации.
головной питон код содержится в Code/Game.py, оттуда и запускается приложение
